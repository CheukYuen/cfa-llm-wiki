# Overview

This is a personal CFA Level I knowledge base. It is not a PDF reader and not
a retrieval system. It is a place to write, in your own words, what you
understand about each topic and concept, and to link those understandings
together.

## How to use this vault

- **Navigation** — open [index.md](index.md). It is auto-generated by
  `tools/build_wiki.py --mode init` and lists every topic, concept, and Q&A
  page currently in the vault.
- **Evolution log** — open [log.md](log.md) to see what tooling has done
  over time (builds, concept extractions).
- **Topics** — `topics/` holds one page per CFA Level I topic area
  (quantitative methods, economics, fixed income, etc.). Each page points at
  the concepts that live underneath it.
- **Concepts** — `concepts/` holds one page per idea. One concept per file.
  Short and densely linked.
- **Q&A** — `qa/` holds open questions the author has asked themselves,
  answered in their own voice, and will revise as their understanding
  sharpens.

## Ground rules (see `schema/AGENTS.md` for the full list)

- PDFs under `raw/` are ground truth. Never edit them.
- `staging/` is disposable. Regenerate it any time.
- `wiki/` is the product. Write here, not anywhere else.
- Never paste PDF text into a wiki page. Paraphrase.
- One concept per page. Link instead of duplicating.
